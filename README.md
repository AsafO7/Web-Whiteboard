# Web-Whiteboard

A MERN project in which you can create a user, open a new drawing room or join an existing room and draw with other users.

## Usage
Rename the .envexample to .env and add your MONGO_URI

### Install dependencies
 **Backend deps** <br>
  ```
  npm install
  ```

 **Frontend deps** <br>
 ```
cd frontend
npm install
```

### Run Server
```
npm run dev
```
